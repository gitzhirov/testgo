# Набор простых комманд git

```
git
	add - добавить изменения в отслеживание гита
	commit - фиксация добавленных изменений в отслеживание
	log - журнал коммитов
	show - просмотр коммита
	push - отправка изменений в удаленный репозиторий

git add <файл/файлы>
git commit
git commit -m "<комментарий>"
git log
git log -n <количество>
git show <коммит>
git push <репозиторий> <ветка>
	git push origin master
git push -u origin master
	git push
```
